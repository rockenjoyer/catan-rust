#### Main Menu Buttons
- Start Game
- Multiplayer

  1. Setup

  4. Robber

The use of generative AI was limited to the [Mistral Ai Le Chat](https://chat.mistral.ai/chatmodel), with multiple versions released during development between November 2025 and February 2026. The specific use of generative AI was strictly limited to error handling of non-descriptive panics or crashes, documentation searching of continously developing libraries with lack of their own documentation.
